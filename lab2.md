## 1. Dictionary of Basic English Terms

- **CLI (Command Line Interface)** — A text-based interface used to interact with the operating system by typing commands.
- **Terminal** — An application that provides access to the command line within a graphical Linux environment.
- **Virtual Terminal** — A text-only Linux mode without a graphical interface, accessed via keyboard shortcuts.
- **Kernel** — The core of the operating system that manages hardware resources.
- **Process** — An instance of a running program.
- **Shell** — A command-line interpreter that processes user commands (e.g., Bash, Zsh).

## 4. Визначення основних понять

- **CLI-режим** — режим роботи ОС, у якому керування здійснюється через текстові команди без графічного інтерфейсу.
- **Термінал на основі GUI** — програма в графічному середовищі, що емулює командний рядок.
- **Віртуальний термінал** — текстова сесія Linux, незалежна від графічної оболонки.

## 1. Робота в графічному режимі ОС Linux (GNOME)

### Основні компоненти GNOME
- **Activities (Огляд)** — перегляд вікон і робочих столів
- **Dash (панель швидкого доступу)** — запуск закріплених програм
- **Пошук** — швидкий пошук програм і налаштувань
- **Workspaces** — динамічні робочі столи
- **Quick Settings** — мережа, звук, яскравість, живлення

---

### 1.2 Запуск програм

**Способи запуску:**
1. Через **Dash**  
2. Через **пошук у Activities**  
3. Через **Show Applications**  
4. Через **Alt + F2** (вікно виконання команд)

---

### 1.3 Вихід з системи та завершення роботи

- **Зміна користувача:** Log Out → екран входу  
- **Перезавантаження:** Restart  
- **Вимкнення:** Power Off
